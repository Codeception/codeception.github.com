Install Phar Executable for PHP 7.x

Download 
wget https://codeception.com/codecept.phardownload

Execute it as php codecept.phar

Install Phar for PHP 5.6

Download 
wget https://codeception.com/php5/codecept.phardownload

Execute it as php codecept.phar

Install Phar Globally 

sudo curl -LsS https://codeception.com/codecept.phar -o /usr/local/bin/codecept
sudo chmod a+x /usr/local/bin/codecept

Execute it as codecept
